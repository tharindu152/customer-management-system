# Customer Management App

### Version

0.1.0

### About The Project

This project is created for a customer management web application.
The REST API for the mentioned web application was coded using undermentioned technologies

- Java Spring
- Hibernate ORM
- Hibernate Validator
- Lombok
- MySQL
